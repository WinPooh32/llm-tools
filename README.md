# llm-tools

A collection of Model Context Protocol (MCP) tools for interacting with files and the terminal.

## Available Tools

### File tools

- `read`: Read a file line by line with line numbers.
- `apply`: Apply changes to a text file.

#### Installation

To install the File tools:

```bash
go install github.com/winpooh/llm-tools/tools/file@latest
```

### Terminal tools

- `exec`: Execute allowed terminal commands.

#### Installation

To install the Terminal Tool:

```bash
go install github.com/winpooh/llm-tools/tools/terminal@latest
```
